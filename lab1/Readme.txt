Задача классификации методом 1NN.

Используется метрический алгоритм классификации 1NN. В задаче используется выборка, содержащая 150 объектов-ирисов:
по 50 объектов каждого из трёх классов. В задаче требуется определить класс любого объекта. 

Решение:
1. Сортируется выборка по расстоянию.
2. Используется метод 1NN, который возвращает класс ближайшего соседа классифицируемого объекта.
3. Рисуется выборка.
4. Рисуются классифицируемые объекты.
